## 版本特定测试 (v2.4)

### 录音回放功能
- 位置: `/free`
- 测试步骤:
  1. 点击"开始"启动麦克风
  2. 进行录音
  3. 查看录音列表
  4. 播放/删除录音

### 目标音辅助线
- 位置: `/free`
- 测试步骤:
  1. 打开目标音开关
  2. 调整滑块选择目标音
  3. 点击试听
  4. 查看可视化器中的绿色参考线

### 道具系统
- 位置: `/sing`
- 测试步骤:
  1. 点击"开始挑战"
  2. 授权麦克风
  3. 查看道具栏 (跳过/提示/重置)
  4. 测试使用道具
